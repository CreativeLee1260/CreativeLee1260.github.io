Hexadecimal uses a six-character code for colors, RGB specifies colors by red, green, and blue values, and HSL describes colors by hue, saturation, and lightness. Hexadecimal is compact, RGB is direct, and HSL is intuitive for color adjustments.

Preferred color code format: Hexadecimal because I could easily understand it and overall it is the most familiar to me.

HWB summary: HWB (Hue, Whiteness, Blackness) is a color model that specifies colors using three values: the hue (the color type), whiteness (the amount of white mixed in), and blackness (the amount of black mixed in).

CMYK summary:The CMYK color model (also known as process color, or four color) is a subtractive color model, based on the CMY color model, used in color printing, and is also used to describe the printing process itself.
