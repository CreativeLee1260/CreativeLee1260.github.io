Working with Positions

Sticky

This is an example of an element with a sticky position. Its position is based on the user's scroll position.

The sticky position property allows an elemnt to stick to a specified position within its containing element when the webpage is scrolled.

Relative

This is an example of an element with a relative position. It can move around a little, depending on the viewport, but it is in a position relative to itself.

Setting top: 200px; and left: 80px; on a sticky element positions it to stick 200 pixels from the top and 80 pixels from the left of its container when scrolled to that point.

Fixed

This is an example of an element with a fixed position. Its position is relative to the viewport.

Setting top: 110px; and right: 80px; on a fixed element positions it 110 pixels from the top of the viewport and 80 pixels from the right edge, meaning it will remain in that position relative to the viewport as the user scrolls.

Absolute

This is an example of an element with an absolute position. It is similar to the fixed position; however, it is positioned relative to the closest parent element.

In CSS, using position: absolute; with bottom: 30px; and right: 40px; places the element 30 pixels up from the bottom and 40 pixels in from the right of its closest parent that has a position set.
