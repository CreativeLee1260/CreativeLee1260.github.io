So for the article selector I used the CSS flexbox method. First I set the display to flex, and then set flex to wrap, and I made the gap 20px, and added a top padding of 2em. For the section selector I set the flex to 1, made the minimum width to 200px, added a solid border to the left and set it a color of DarkSeaGreen. I added a pseodo-class to the section that removes border to the first child.
